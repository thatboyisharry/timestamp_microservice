Timestamp Microservice

How to use this API:

  1. A request to /api/:date? with a valid date should return a JSON object with a unix key that is a Unix timestamp of the input date in milliseconds and a utc key that is a string of the input date.

    example 1

    A request to /api/2015-12-25 should return
    {"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}


    example 2

    A request to /api/1451001600000 should return
    {"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

  2. An empty date parameter should return the current time in a JSON object with a unix key and a utc key

  3. If the input date string is invalid the api returns a JSON object having the structure
    {error:"Invalid Date}"
